## Appendix C: Urdu Glossary & Localization

This textbook serves the **PIAIC community in Pakistan**. This appendix provides Urdu translations of key terms to help Urdu-speaking engineers understand the technical concepts.

### How to Use This Glossary

- **English term** → Romanized Urdu → اردو (Urdu script)
- Read left-to-right for English, right-to-left for Urdu
- Use these terms when discussing with Urdu-speaking colleagues
